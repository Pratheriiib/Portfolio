Trey's Web Maps

This is a portfolio displaying some of the web mapping pages and applications that I have made throughout my career.

US Power Plants

This is a map I made at UK that shows all of the power plants in the United States that are generate power using Solar, Wind, or Hydro power.

Critical Habitat

This is a project from my Design for Interactive Web Mapping course. It is a simple webpage stylistically, however it contains a map I built using the Leaflet javascript mapping library highlighting areas in Kentucky that are listed as Critical Habitat under the Endangered Species Act

Prescribed Burning at Berea in 2019

This is a map that I made on ArcGIS Desktop that is an overview of the ~300 Acres that we put into our prescribed burning program at Berea in 2018 & 2020

Bars and Restaurants Near Lexington Parks

This web page features a map made in QGIS using Open Street Maps Data. This map shows Streets, Buildins, and Parks in Lexington. In addition, It highlights any bars or restaurants within one mile (walking distance) of those parks.

ArcGIS Online Web Map for Collaborative Study

This is a simple but practical web map that I built on ArcGIS online in order to share the design of a multi-year study that we are undertaking with the folks at the University of Louisville showing the effects of Prescribed Fire on the land.

Classic Raisz Map Overlaid with Appalachian Trail

This project is a web page displaying a classic Erwin Raisz landforms map that has been georeferenced in QGIS and then overlaid with a shapefile of the Appalachian Trail.

About me

Hi! I'm Trey Prather and I am a forester with a diverse mapping skill set. I am comfotable with the ESRI platform of software as well as QGIS. In addition to traditional geospatial analysis, my time in the New Mapping Plus program at UK has taught me everything from database management using PostGIS, PostgreSQL and Spatialite, to web mapping and site design using HTML, CSS, and Javascript. I also have learned to use such web mapping databases such as Carto, D3, Leaflet, and MapBox!

When I'm not in the woods or behind a computer screen you can catch me hanging out with my wife Kennedy and my daughter Juniper. I also enjoy playing disc golf, fantasy football, and softball with my friends. My other hobbies include enjoying the outdoors through hiking or fishing, and playing guitar!
